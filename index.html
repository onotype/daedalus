Daedalus: a static HTML
project boilerplate

Quick and dirty boilerplate for when frameworks are just overkill.

Copy this folder and set it up on VS Code and Prepros.
Prepros config is available in Obsidian.

  • Target CSS directory: ../../dist/css
  • Target JS directory: ../../dist/js
    • Make sure to enable automatic processing for js files
    • This isn't enabled by default
